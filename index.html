「Webコンテンツ作成」の成果

本テーマについて

先行する情報工学の専門科目の基礎知識を応用して実践的な能力を修得することが目的。到達目標は実験の一連の流れを実践でき、グループワークを円滑に実施し、モーション解析・編集ができ、HTML等を用いたWebページ作成ができること。

プロジェクト課題について

検索機能をつけるのに難航して、調べ物にあまりにも時間をかけてしまった。しかしその成果か、納得のいくページが作成できたと思う。また今までの授業一個一個は理解できていてもまとめると理解できていない点があったり応用ができない部分が多々あり、復習が必須だと感じた。デザイン面は苦戦することなく自身の好きな雰囲気にできたので満足したが、時間があればもう少し細かくやってみたかった。

project.html

数当てゲーム

テキストを入力できるボックスを利用し、とある変数にランダムに入力された数字を当て、またその結果が答えより大きいのか小さいのか表示させるゲームを作成した。第3回の演習ではその結果をコンソールに表示したが、最終的には結果を実際にページに文字として表すことに成功した。制作の中で、ページに文字を表示させる方法を知り、javaではなくjavascriptのif文を利用し、それらを組み合わせる方法を学んだ。またイベントを起こすfunctionを作成し、テキストボックスの中にプレイヤーが入力した数値を作成したプログラムの中に代入し、その結果をまた文章に戻す方法も学んだ。

第3回演習

仮の予想として４をおき、ランダムに正解とされる１〜１０の数値と同じか、それより大きいかまた小さいかの結果をコンソールに表示させた。 数当てゲーム

各授業回の学習事項

第1回 WWW と HTML

HTMLとは何かを学んだ。要素の名前やデバッグの方法を知った。演習ではHTMLの様々な要素について実践で学び、画像の貼り方や見出しの付け方を知った。


演習の成果：演習1-4:HTMLのさまざまな要素

第2回 CSS による Webページの装飾

CSSの基本的な説明および使い方を学んだ。具体的には、HTMLに適応する方法や文字に装飾をするために使う属性名、またボックスの作り方やグラデーションの設定方法を学び演習で理解を深めた。
演習の成果：演習2-3:project.htmlのCSS作成

第3回 JavaScript の基本

JavaScriptの基礎的な内容を学んだ。Javaとの大きな違いは変数にあると知った。一律letで定義できるJavaScriptはエラーがわかりにくいデメリットもありながら比較的簡単に覚えられそうだった。
演習の成果：演習3-6

第4回 DOM と JavaScript

JavaScriptとHTMLを利用し、ボタンを押したらページが変化するイベントのやり方を学んだ。演習ではテキストボックスに入力した二つの数字を足し、ボタンを押したら答えを表示させるページを作成した。
演習の成果:演習4-10

第５回 イベント処理

今まで見てきた構文ではないもので、コンソールではなくページ上に文字列を表示させる方法を学んだ。演習では表示ボタンを押したらキャンパスの住所と大学の学科一覧が表示されるページを作成した。
演習の成果:演習5-7

工夫点、苦心点、強調点，感想

私は「情報メディア実験Webコンテンツ作成」の授業を通じて、HTML・JavaScript・CSSというWeb制作に欠かせない基本技術の使い方を学んだ。それまでは「HTML」という言葉を聞いたことがある程度で、Webページがどのように作られているかについては全く知らなかった。最初は、専門用語の多さや構文の違いに戸惑うことも多く、本当にゼロからのスタートだった。しかし、授業を通じて順を追って学習する中で、HTMLによってページの構造を作り、CSSで見た目を整え、JavaScriptでページに動きを加えるという基本的な仕組みを少しずつ理解できるようになっていった。 最終課題として、グルメ情報を紹介するWebサイトを制作することになった。自分でページ構成を設計しコンテンツを作り上げていく作業は、とても楽しく充実したものだった。中でも、CSSを用いたデザイン調整には特に力を入れ、色使いやレイアウトのバランスに気を配ることで、自分なりに満足のいくビジュアルに仕上げることができた。一方で、反省点もある。例えば、ユーザーが入力できるテキストボックスの配置や大きさ、見た目などについては、もう少し工夫の余地があったと感じている。使いやすさや視認性に配慮した設計ができていれば、より完成度の高いWebサイトになったはずだ。 今回の授業を通して、Webサイトを作ることの楽しさと奥深さを実感した。最初はHTMLやCSS、JavaScriptといった言葉もよく知らない状態だったが、授業を受けながら少しずつ使い方を覚えていった。そして、自分の考えた内容を画面に形として表せるようになって、Web制作への興味がさらに高まった。ページの作り方や色の使い方、文字の大きさや配置、ボタンの並べ方などを工夫することで、見た目や使いやすさが大きく変わることに驚いた。失敗点もあったが、自分で試行錯誤しながら完成させたときの達成感はとても大きく、さらに興味が湧いてきた。 これからは、今回学んだ基本的な技術をもとにして、さらに新しいことにもチャレンジしていきたい。たとえば、スマートフォンでも見やすくなるような画面の作り方や、動きのあるページの作り方、なども学んでみたい。そして、将来は自分の好きなことを発信できるオリジナルのWebサイトを作って、いろんな人に見てもらえるようにしたい。今回の授業で学んだことを糧として、これからももっと勉強し自分だけのWebサイトを作れるようになりたい。
