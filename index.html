数当てゲーム

テキストを入力できるボックスを利用し、とある変数にランダムに入力された数字を当て、またその結果が答えより大きいのか小さいのか表示させるゲームを作成した。第3回の演習ではその結果をコンソールに表示したが、最終的には結果を実際にページに文字として表すことに成功した。制作の中で、ページに文字を表示させる方法を知り、javaではなくjavascriptのif文を利用し、それらを組み合わせる方法を学んだ。またイベントを起こすfunctionを作成し、テキストボックスの中にプレイヤーが入力した数値を作成したプログラムの中に代入し、その結果をまた文章に戻す方法も学んだ。

第3回演習

仮の予想として４をおき、ランダムに正解とされる１〜１０の数値と同じか、それより大きいかまた小さいかの結果をコンソールに表示させた。 数当てゲーム

各授業回の学習事項

第1回 WWW と HTML


演習の成果：演習1-4:HTMLのさまざまな要素

第2回 CSS による Webページの装飾

CSSの基本的な説明および使い方を学んだ。具体的には、HTMLに適応する方法や文字に装飾をするために使う属性名、またボックスの作り方やグラデーションの設定方法を学び演習で理解を深めた。
演習の成果：演習2-3:project.htmlのCSS作成

第3回 JavaScript の基本

JavaScriptの基礎的な内容を学んだ。Javaとの大きな違いは変数にあると知った。一律letで定義できるJavaScriptはエラーがわかりにくいデメリットもありながら比較的簡単に覚えられそうだった。
演習の成果：演習3-6

第4回 DOM と JavaScript

JavaScriptとHTMLを利用し、ボタンを押したらページが変化するイベントのやり方を学んだ。演習ではテキストボックスに入力した二つの数字を足し、ボタンを押したら答えを表示させるページを作成した。
演習の成果:演習4-10

第５回 イベント処理

今まで見てきた構文ではないもので、コンソールではなくページ上に文字列を表示させる方法を学んだ。演習では表示ボタンを押したらキャンパスの住所と大学の学科一覧が表示されるページを作成した。
演習の成果:演習5-7
